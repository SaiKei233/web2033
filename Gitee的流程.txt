1.  下载安装了 Git ，添加用户名和邮箱
2.  注册了 Gitee 平台
3.  添加公钥到Gitee 平台  > ssh-keygen -t ed25519 -C '你的邮箱' 
4. 克隆远端仓库   git clone git@gitee.com:yulonny/web2203_repository.git
5. git add .   将修改的所有数据 添加到 本地 暂存区
6. git commit -m '日志记录'  将暂存区的 数据 提交到 本地 仓库
7. git push origin master  将本地的 提交版本 推送到 服务器仓库
8. git pull origin   拉取服务器 最新的版本到 本地仓库
